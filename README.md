# 小费分配助手

一个纯静态网页版本的小费分配计算器，可以直接上传到 GitHub Pages。

## 功能

- 输入总小费 Tip Total
- 设置 Boss / Kitchen 百分比
- 输入每个人的工时
- 自动计算：
  - Boss 小费
  - Kitchen 小费
  - Subtotal
  - Total Hour
  - Tips / h
  - 每个人分到的小费
  - Remaining

## 本地打开

直接双击 `index.html` 即可。

## 上传到 GitHub Pages

1. 新建 GitHub repository
2. 上传这几个文件：
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. 进入 repository 的 `Settings`
4. 打开 `Pages`
5. Source 选择：`Deploy from a branch`
6. Branch 选择：`main`，folder 选择 `/root`
7. 保存后等待 1-2 分钟

GitHub 会生成一个网页链接。
